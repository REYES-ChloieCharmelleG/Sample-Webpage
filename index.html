What's HTML?

HTML (HyperText Markup Language) is a standard markup language used to create web pages. It provides the structure and content of a webpage.

Basic Elements of HTML

  • h1 to h6: Heading elements, with h1 being the largest and h6 the smallest.
  • p : Defines a paragraph of text.
  • a : Creates a hyperlink, linking to another document or a specific part of the current document using the href attribute.
  • strong : Renders text with strong importance, typically displayed in bold.
  • em : Renders text with emphasis, typically displayed in italics.
  • span : An inline container for styling or scripting a small portion of text.
